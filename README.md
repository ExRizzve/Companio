# Companio

Летающий компаньон с головой любого игрока Minecraft.

[![Minecraft](https://img.shields.io/badge/Minecraft-26.1.2-62B47A?style=flat-square)](https://www.minecraft.net/)
[![Fabric](https://img.shields.io/badge/Loader-Fabric-DBD0B4?style=flat-square)](https://fabricmc.net/)

Companio добавляет небольшого визуального спутника, который летает рядом с игроком, следит за ним и иногда отвлекается на собственные дела. Компаньон может носить скин друга, имя над головой и выполнять несколько анимаций.

Мод полностью клиентский. Серверу и другим игрокам ничего устанавливать не нужно, но видеть компаньона будет только его владелец.

## Возможности

- загрузка скинов по нику через сервисы Mojang;
- стандартный Стив, если ник не указан;
- плавный свободный полёт и быстрое возвращение к владельцу;
- случайные вращения и полёт вокруг игрока;
- имя над компаньоном через `TextDisplay`;
- Tab Completion для ников игроков и типов событий;
- русский и английский интерфейс;
- конфигурация с проверкой некорректных значений.

## Команды

| Команда | Действие |
|---|---|
| `/companio create` | Создать компаньона со стандартной головой Стива |
| `/companio create <ник>` | Создать компаньона со скином указанного игрока |
| `/companio name <имя>` | Установить имя длиной до 32 символов |
| `/companio name clear` | Убрать имя |
| `/companio event` | Запустить случайное событие |
| `/companio event spin` | Выполнить оборот на 360° |
| `/companio event twirl` | Выполнить продолжительное вращение |
| `/companio event orbit` | Полететь вокруг владельца |
| `/companio reload` | Перезагрузить конфигурацию |
| `/companio remove` | Убрать компаньона |

## Установка

Понадобятся:

- Minecraft Java Edition `26.1.2`;
- Fabric Loader `0.19.2` или новее;
- Fabric API для `26.1.2`.

Скачайте JAR из раздела [Releases](../../releases) и положите его в папку `.minecraft/mods`. Companio не требуется устанавливать на сервер.

## Конфигурация

После первого запуска появится файл:

```text
.minecraft/config/companio.json
```

В нём можно изменить высоту полёта, радиус свободного движения, дистанцию возвращения, скорость и ускорение. Значения вне безопасного диапазона автоматически исправляются при загрузке.

## Сборка из исходников

Для разработки требуется JDK 25.

```bash
./gradlew build
```

Готовый мод появится в `build/libs`:

```text
companio-1.0.1+26.1.2.jar
```

## Важно

Companio - визуальный клиентский мод. Компаньон не существует на стороне сервера, не имеет хитбокса, не взаимодействует с миром и не даёт игровых преимуществ.
