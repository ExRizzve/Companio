# Companio

Небольшие летающие компаньоны со скинами игроков Minecraft.

Companio добавляет до шести визуальных спутников, которые самостоятельно летают рядом с владельцем. Для каждого можно выбрать скин игрока, установить отдельное имя и настроить характер движения.

Компаньоны плавно исследуют пространство, обходят блоки, держат дистанцию друг от друга и ускоряются, если отстают от владельца.

Мод полностью клиентский. Устанавливать его на сервер не нужно, но компаньонов видит только игрок с установленным модом.

## Что изменилось в 1.2

Версия `1.0.1` была тестовой и использовалась для проверки основной идеи мода. В `1.2` поведение и внутренняя структура Companio были заметно переработаны.

- теперь можно создать до шести компаньонов одновременно;
- каждый компаньон самостоятельно выбирает маршрут;
- убраны случайные вращения и полёт по кругу;
- добавлено быстрое возвращение к отставшему владельцу;
- компаньоны стараются не задевать блоки и не проходить сквозь них;
- компаньоны держат небольшую дистанцию друг от друга;
- каждому компаньону можно назначить отдельное имя;
- добавлено удаление по номеру;
- настройки полёта можно менять командами без перезапуска;
- для команд и значений работает Tab Completion.

## Улучшенное поведение

У каждого компаньона есть собственная цель, скорость и момент смены направления. Благодаря этому они не двигаются одной группой и не пытаются постоянно кружить вокруг игрока.

Во время полёта компаньоны:

- выбирают свободные точки на разной высоте;
- плавно ускоряются и тормозят;
- поворачиваются по направлению движения;
- скользят вдоль препятствий или поднимаются над ними;
- меняют маршрут, если путь полностью закрыт;
- быстро догоняют владельца при большом отставании;
- мягко расходятся, когда подлетают слишком близко друг к другу.

## Команды компаньонов

| Команда | Действие |
|---|---|
| `/companio create` | Создать компаньона со стандартной головой Стива |
| `/companio create <ник>` | Создать компаньона со скином указанного игрока |
| `/companio name <номер> <имя>` | Установить отдельное имя длиной до 32 символов |
| `/companio name <номер> clear` | Убрать имя выбранного компаньона |
| `/companio remove <номер>` | Удалить выбранного компаньона |
| `/companio remove all` | Удалить всех компаньонов |
| `/companio reload` | Повторно загрузить конфигурацию из файла |

После создания в чате появляется номер компаньона. Этот номер используется в командах имени и удаления. После удаления одного компаньона оставшиеся номера сдвигаются без пропусков.

## Команды конфигурации

Настройки применяются сразу ко всем компаньонам и сохраняются в `companio.json`.

| Команда | Диапазон | Действие |
|---|---:|---|
| `/companio config distance <значение>` | `1–12` | Изменить дальность свободного полёта |
| `/companio config height <значение>` | `1.5–6` | Изменить высоту полёта |
| `/companio config speed <значение>` | `0.04–0.2` | Изменить обычную скорость |
| `/companio config follow-speed <значение>` | `0.25–0.8` | Изменить скорость догоняния |
| `/companio config smoothness <уровень>` | `1–10` | Изменить плавность ускорения и торможения |
| `/companio config sharpness <уровень>` | `1–10` | Изменить резкость поворота |
| `/companio config reset` | — | Вернуть стандартные настройки |

Для названий команд, номеров компаньонов и рекомендуемых значений доступны подсказки по Tab. При этом допустимые значения можно вводить вручную.

Пример настройки спокойного и плавного полёта:

```text
/companio config distance 8
/companio config speed 0.1
/companio config follow-speed 0.5
/companio config smoothness 8
/companio config sharpness 6
```

## Файл конфигурации

После первого запуска создаётся файл:

```text
.minecraft/config/companio.json
```

Его можно редактировать вручную или через команды. Некорректные значения автоматически заменяются безопасными при загрузке.

## Установка

Понадобятся:

- Minecraft Java Edition `26.1.2`;
- Fabric Loader `0.19.2` или новее;
- Fabric API для `26.1.2`.

Скачайте JAR из раздела [Releases](../../releases) и положите его в папку `.minecraft/mods`. Устанавливать Companio на сервер не требуется.

## Сборка из исходников

Для разработки требуется JDK 25.

```bash
./gradlew build
```

Готовый мод появится в `build/libs`:

```text
companio-1.2+26.1.2.jar
```

## Важно

Companio — визуальный клиентский мод. Компаньоны не существуют на стороне сервера, не имеют хитбокса, не взаимодействуют с игровым миром и не дают преимуществ в игре.
